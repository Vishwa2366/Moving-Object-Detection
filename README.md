# Moving-Object-Detection
# Moving Object Detection using OpenCV

## Project Overview
This project detects moving objects in a video stream using computer vision techniques. The system compares frames from a webcam feed and highlights regions where motion is detected.

##  Technologies Used
- Python
- OpenCV
- Imutils

##  Features
- Real-time motion detection
- Identifies moving objects in video frames
- Displays bounding boxes around moving areas

##  How to Run
1. Install required libraries:
   pip install opencv-python
   pip install imutils

2. Run the script:
   python motion_detection.py

##  Output
The system opens a webcam and detects motion by highlighting moving regions.

## Learning Outcome
- Frame differencing techniques
- Motion detection algorithms
- Real-time video processing
